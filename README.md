# node-api-template
Nodejs express api template

Uses:
- express
- body-parser
- cors
- morgan
- winston
- dateformat

Instructions:
1. git clone https://github.com/manjitsagoo/node-api-template.git
2. cd node-api-template
3. npm install
4. node server.js
5. browse to http://localhost:3000/api/version
